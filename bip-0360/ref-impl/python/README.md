
# p2tsh python implementation and test vectors

This python project contains a simple implementation of some essential function to form and spend both Pay-to-Taproot (p2tr) and Pay-to-Tapscript-Hash (p2th) transaction outputs.

# How to Use

This project was developed with `Python 3.13.x`, but any modern interpreter should do just nice.

# Tests

To run unit tests,

``` bash
source test.sh
```

# Examples

Other instructive and end-to-end examples can be found in `examples/`
